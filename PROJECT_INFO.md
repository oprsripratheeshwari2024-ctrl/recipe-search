# 🍲 Recipe Search App

## 📌 Project Description
This is a React-based Recipe Search Application that allows users to search recipes by ingredient using TheMealDB API.

---

## 🚀 Features
- Search recipes by ingredient
- Display recipe images and names
- API integration (TheMealDB)
- No results handling
- Responsive card layout
